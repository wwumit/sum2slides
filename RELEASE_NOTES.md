## 📦 安装方式

### 通过ClawHub安装（推荐）
```bash
openclaw skill install sum2slides
```

### 通过pip安装
```bash
pip install sum2slides
```

### 从源码安装
```bash
通过 OpenClaw 安装技能
cd sum2slides
pip install -e .
```

## 🚀 快速开始

### 基本使用
```bash
# 从文本生成幻灯片
sum2slides "项目总结：完成核心功能开发，通过测试验证" --output demo.pptx
```

### 查看演示
```bash
python scripts/demo.py
```

### 阅读教程
```bash
cat TUTORIAL.md
```
