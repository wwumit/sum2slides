## ☁️ 云文档上传说明

### 准备工作
1. **用户ID**: `ou_65416d104b91e112b6ab125962d4dbe1` (owner_open_id)
2. **内容文件**: `cloud_doc/sum2slides_cloud_doc.md` (4649字符)
3. **文档标题**: "Sum2Slides v1.0.1 演示文档"

### 上传方法

#### 方法1: 创建新文档（推荐）
```json
{
  "action": "create",
  "title": "Sum2Slides v1.0.1 演示文档",
  "owner_open_id": "ou_65416d104b91e112b6ab125962d4dbe1"
}
```

#### 方法2: 写入现有文档
```json
{
  "action": "write",
  "doc_token": "文档token",
  "content": "云文档内容..."
}
```

### 上传步骤
1. **打开飞书文档技能** (feishu-doc)
2. **使用create动作**创建新文档
3. **确认owner_open_id**设置为用户ID
4. **文档创建后获取doc_token**
5. **使用write动作**写入内容（或将内容包含在create中）

### 注意事项
1. **权限**: 确保owner_open_id正确设置，以便用户拥有完全访问权限
2. **文件夹**: 如果需要指定文件夹，可以添加folder_token参数
3. **格式**: 内容为Markdown格式，飞书文档会自动解析
4. **链接**: 文档中包含的GitHub链接等会自动转为可点击链接
