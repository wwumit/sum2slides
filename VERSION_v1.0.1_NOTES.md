## 🚀 安装方式

### 通过 OpenClaw 安装（推荐）
```bash
# 安装技能
openclaw skill install sum2slides

# 验证安装
openclaw skill list | grep sum2slides
```

### 本地安装
```bash
# 进入技能目录
cd skill-package/

# 创建技能包
tar czf sum2slides-1.0.1.skill .

# 安装技能包
openclaw skill install sum2slides-1.0.1.skill
```

## 🎯 使用说明

### 基本使用
```bash
# 在OpenClaw中调用
@sum2slides 请将会议纪要转为幻灯片

# 或使用命令模式
/open sum2slides --input notes.txt --output slides.pptx
```

### 典型使用场景
- **会议纪要转幻灯片**: 快速生成会议总结演示
- **项目报告生成**: 自动创建项目进度报告
- **学术汇报制作**: 帮助教师和学生制作课件
